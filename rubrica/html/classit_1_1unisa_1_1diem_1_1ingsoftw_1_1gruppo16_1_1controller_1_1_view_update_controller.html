Detailed Description

@biref Classe controller per la gestione delle scene in JavaFX.

Questa classe permette di gestire l'aggiornamento delle scene dell'applicazione. Esiste solo una istanza per questa classe (Singleton pattern). Oltre a caricare le scene da file FXML, la classe permette di inserire anche file css per personalizzare l'applicazione. @Class ViewUpdateController
